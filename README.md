## Selected projects

- [TableLite](https://tablelite.nl) - restaurant reservation system with timeline, calendar, floor plan, CRM, and white-label booking flows.
- [Blackjack](https://blackjack.aldo.al) - multiplayer blackjack with authoritative server logic and in-game purchases.
- [SuperInvestor](https://investor.aldo.al) - note-taking app for SEC filings built with Blazor Server and PostgreSQL.
- [Trap Or Value](https://stocks.aldo.al) - financial guessing game built with Blazor Server, FastAPI, and a shared SQLite dataset.
- [QuizPlus](https://www.quizplus.io) - quiz game with multiplayer and AI-generated content.
- [StockPortfolio.me](https://www.stockportfolio.me) - portfolio sharing app with donut chart visualizations.
